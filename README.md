# GymLeads India

Auto-generated by ETS-AI 10x.

## Idea Summary
A SaaS platform to help small local gyms in India capture, track, and close leads from WhatsApp and Instagram.

## Problem Statement
Small local gyms in India lack an efficient system to manage and convert leads generated from popular platforms like WhatsApp and Instagram.

## Target Persona
Indian small gym owner

## MVP Features
- WhatsApp Business API integration for lead capture and communication
- Instagram Graph API integration for lead capture and communication
- Unified Lead Inbox for all incoming messages from WhatsApp/Instagram
- Lead Profile View (contact details, source, interaction history)
- Lead Status Tracking (New, Contacted, Follow-up, Converted, Lost)
- Ability to add notes and custom tags to leads
- Basic Follow-up Reminders and scheduling
- Dashboard with key metrics (total leads, conversion rate, leads by source)
- User authentication and basic gym profile setup

## Tech Stack
Backend: Python (Django/FastAPI) for rapid development and ML integration. Frontend: React.js/Vue.js for a responsive UI. Database: PostgreSQL for robust data management. Cloud: AWS/GCP for scalability and managed services. Messaging APIs: WhatsApp Business API, Instagram Graph API. ML: Scikit-learn for initial lead scoring.
